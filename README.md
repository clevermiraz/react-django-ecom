
## Installation

Step 1: Clone the Repository

Step2: Go to Part 1 with command:
```bash
  git checkout part1
```

Step3: then go there where package.json live

Step4: the run this cmd:
```bash
  yarn
```
Or

```bash
  yarn install
```
Or

```bash
  npm install
```

that's it 🙂🤞


## Run React Frontend

```bash
  yarn start
```

Or 

```bash
  npm start
```


Recomend using Yarn🤞
